Week 02 Stylings and Semantics

* Learned how to apply Semantics in order to better organize sections of information in a website.

* Learned how to use CSS to change the adjectives of a webpage, including font and colour.

* Learned how to use ID atttribute & Class attribute.

- ID attribute: can only be used for one element.

- Class attribute: can be used for multiple elements.

* Learned how to use Pseudo classes.

Week 03 Tags and Positioning

* Learned how to use form, input tag along with label.

* Learned positioning commands such as position = static/relative/absolute/fixed.

* Justify-content: (flex-start/flex-end/center/space-between/space-around).

* Align-items: (flex-start/flex-end/center).

* Flex-direction: (row/row-reverse/column/column-reverse).

Week 04 Responsive Images

* Width: to change the size of the image.

* Max-width: limits how much the image can be scaled.

* padding: gives a border around the picture.

* background-size: cover (fits the image in a box so that it is fully shown even when there is another picture next to it.)

* flex-wrap: puts items to a new row when there isn't enough space.

* flex-grow: (Value/inherit)

- Value: scales images proportional to the value.

- Inherit: items are of the same size to the parent item.

* flex-basis: scales images with regards to screen size.
